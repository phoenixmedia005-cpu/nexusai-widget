# 🔥 Blaze256 — Client Activation Checklist

Use this every time a new client submits payment on blaze256.com.

---

## STEP 1 — Verify Payment ✅

- [ ] Open MTN MoMo or Airtel Money app
- [ ] Check that **TXN ID** matches what the client submitted
- [ ] Confirm the **amount** matches their plan:
  - Starter: **UGX 300,000**
  - Growth: **UGX 500,000**
- [ ] Confirm the **sender name/number** matches their phone

> ⚠️ If TXN ID doesn't match or amount is wrong — WhatsApp the client before proceeding.

---

## STEP 2 — Register Their Widget 🛠️

Go to: **https://6a090d6e0c6512ce0c81a246.base44.app** (NexusAI Dashboard)

- [ ] Click **"Deploy"** tab
- [ ] Fill in:
  - Company Name: *(client's business)*
  - Website: *(their site URL)*
  - Industry: *(e.g. Restaurant, Real Estate, Clinic)*
  - Agent Name: *(e.g. "Blaze" or custom)*
  - Handoff Email: *(client's email)*
- [ ] Click **"Register Widget"**
- [ ] Copy the **Widget ID** and **Embed Code** generated

---

## STEP 3 — Send Embed Code to Client via WhatsApp 📲

Copy this message and fill in the blanks:

```
Hi [NAME]! 🔥

Your Blaze256 AI Sales Agent is now LIVE!

Here's your widget embed code — paste it just before the </body> tag on your website:

<script src="https://blaze256.com/widget.js" data-widget-id="[WIDGET_ID]"></script>

That's it! Your AI agent will start capturing leads immediately.

If you need help installing it, send me your website login and I'll do it for you.

Welcome to Blaze256! 🚀
— Phoenix MEDIA
```

---

## STEP 4 — Log the Client in Dashboard 📊

- [ ] Open **Leads** tab in NexusAI Dashboard
- [ ] Find their lead record (auto-created from payment form)
- [ ] Update status to: **"Active Client"**
- [ ] Add notes: Plan, payment date, widget ID

---

## STEP 5 — Follow Up in 48 Hours 🔔

Send a quick WhatsApp:

```
Hi [NAME]! Checking in — did you manage to install the widget on your site?
If you need any help, I'm here. 😊
— Blaze256 Team
```
